Step-1

The objective is to determine when an upper triangular matrix is nonsingular.

Let be an upper triangular matrix.

Then, all the entries below the main diagonal are zeros.

A matrix A is nonsingular if its determinant is not equal to zero.

For any triangular matrix, the determinant is the product of the diagonal entries.

Therefore, the determinant of U is the product of all the elements in the main diagonal and it is not equal to zero only if all the diagonal entries are nonzero.

That is, the diagonal entries must be all nonzero.

Hence, an upper triangular matrix is nonsingular if all the diagonal entries of the matrix are nonzero.
